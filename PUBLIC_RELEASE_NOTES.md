# Public Release Notes

## Elite Review Bench v1.0.2

This is a standalone public-release advisory-review framework for structured decision support, board-style second opinions, dissent memos, risk identification, and expert-review detection.

## Release Purpose

Create a portable, evidence-constrained review framework that any user can upload into a new AI instance without requiring private project memory, proprietary architecture, institutional affiliation, or internal operating doctrine.

## Included Capabilities

- Board-style advisory review.
- Go / conditional-go / no-go / defer recommendations.
- Evidence-tier separation.
- Verified-fact versus assumption separation.
- Risk register generation.
- Dissent memo generation.
- Expert-review trigger detection.
- PMO and execution-risk review.
- Capital-risk and governance review.
- Pre-mortem review.
- Claude Code self-contained instruction support.

## Public-Release Cleanup

- Uses neutral public product identity: Elite Review Bench.
- Removes private architecture dependencies.
- Removes affiliation-implying labels.
- Adds explicit no-third-party-affiliation language.
- Frames the product as advisory decision support, not execution authority.
- Strengthens professional-advice, approval-authority, warranty, and user-responsibility limitations.
- Blocks final professional advice in regulated or high-consequence domains without expert review.
